## 有什么用？

生成hdu杭电的大职课程，宣讲会签到码



## 为什么要这样做？

很多宣讲会或则招聘会压根没有提供签到码，一个一个去参加很浪费时间，不如去做点其他有意义的事情



## 如何使用？

修改start_id和count的值来决定宣讲会的时间以及数量

在任意python环境下运行，运行代码后，会在路径下生成一个pic文件夹，文件夹内就是宣讲会签到码




### 如何获取start_id?

前往杭电就业网[杭州电子科技大学24365就业数智平台](https://job.hdu.edu.cn/)，找到下方的校园宣讲会专栏，随便找一个点进去

比如[浙江明贺钢管有限公司](https://job.hdu.edu.cn/teachin/view?id=136995)这种，那个id就是start_id

一般宣讲会是按照时间顺序排列的
